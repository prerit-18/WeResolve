# WeResolve

## -------CURRENTLY BUILDING-------

## About

WeResolve is an AI-powered civic issue reporting platform that helps citizens report local community problems such as potholes, garbage, broken streetlights, water leakage, and more.

The platform connects citizens, volunteers, and administrators to ensure issues are resolved efficiently. Volunteers can accept reported issues, upload proof after resolving them, and earn **Social Impact Credits** for their contributions. Administrators verify completed tasks before rewarding contributors.

The goal of WeResolve is to encourage community participation and make cities cleaner, safer, and more responsive through technology.

## Vision

To build a platform where every citizen can contribute to solving real-world problems and create a positive impact in society.

## Core Features

- Report civic issues with images and location
- AI-assisted issue categorization and prioritization
- Volunteer-based issue resolution
- Admin verification system
- Social Impact Credits for contributors
- Real-time issue tracking and analytics

## Tech Stack

- React
- FastAPI
- MongoDB
